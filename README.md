## Usage

Download the ZIP file or ```git clone``` to a directory.

Open _passwords.csv_ and begin adding your account information.

```Start > Run```....Oh wait...its Linux...just open a ```terminal```

```cd``` to the directory _puu2.sh_ is located.

```./puu2.sh```
